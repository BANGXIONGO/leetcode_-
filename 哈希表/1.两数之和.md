### 构造哈希（enumerate） 
哈希表的使用， python使用字典{}记录列表的序号的对值（列表是一个特殊的哈希表）
通过enumerate获得哈希

哈希表最擅长的是==索引==， 所以要利用哈希表就要将两数之和变为求一数之差target- nums [ j ]

```python
class Solution:

    def twoSum(self, nums: List[int], target: int) -> List[int]:

        idx = {}

        for j, x in enumerate(nums):

            if target - x in idx:

                return [idx[target - x], j]

            idx[x] = j

        return idx
 ```
