### 哈希表构造
这题难点在哈希表构造，哈希表的key是分类字符的关键，找出单词互为异位词的程序表达是关键
单词互为异位词的原因是单词排序后都相同，所以可以将排序后的单词作为key

```python
class Solution:

    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:

        d =defaultdict(list)

        for word in strs:

            sortde_word = ''.join(sorted(word))

            d[sortde_word].append(word)

        return list(d.values())
```
