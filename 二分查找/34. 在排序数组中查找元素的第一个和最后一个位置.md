二分查找在寻找数组中大于/ 大于等于/小于 /小于等于每个数target的妙用

```python
def lower_bound(self, nums: List[int], target: int) -> int:
        left, right = -1, len(nums)  # 开区间 (left, right)
        while left + 1 < right:  # 区间不为空
            mid = (left + right) // 2
            # 循环不变量：
            # nums[left] < target
            # nums[right] >= target
            if nums[mid] >= target:
                right = mid  # 范围缩小到 (left, mid)
            else:
                left = mid  # 范围缩小到 (mid, right)
        # 循环结束后 left+1 = right
        # 此时 nums[left] < target 而 nums[right] >= target
        # 所以 right 就是第一个 >= target 的元素下标
        return right

```
1. 寻找大于target --->lower_bound(nums , target-1)+1
2. 寻找大于等于target --->lower_bound(nums , target)
3. 寻找小于target --->lower_bound(nums , target+1)-1
4. 寻找小于等于target --->lower_bound(nums , target-1)+1
