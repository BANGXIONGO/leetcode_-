在多维数组中使用二分查找 ， 注意化繁为简，利用一维中mid索引转化为多维中的mid索引 ， 去寻找所谓的中位数x
 
**x =matrix[mid// n ]   [ mid %n]对于mid值在矩阵的对于索引值， 用于比较大小**

```python 
class Solution:

    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:

        m , n  = len(matrix) , len(matrix[0])

        left  , right = -1  , m*n#使用双开区间

        while left +1  <right:

            mid = (left +right) //2

            x =matrix[mid//n][mid %n]#对于mid值在矩阵的对于索引值， 用于比较大小

            if x == target :

                return  True

  

            if x >target:

                right = mid

  

            if x < target:

                left = mid

        return False

```
